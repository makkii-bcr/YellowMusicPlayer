Ymplayer.initPlayingOption

Midiの各再生オプションをデフォルト値に戻します

Ymplayerクラスを new で生成したときに、このメソッドが自動的に呼び出されます
(コンストラクタで呼び出しているため)
※ このメソッドの後に、updatePlayInfo()を呼び出すことで、MusicPlayerに反映されます

書式

initPlayingOption()
下記のように変数がデフォルト値になります
  • midiSkipBeginning = 1;
  • midiSkipEnding = 0;
  • midiMasterPitch = 0;
  • midiNormalize = 0;
  • midiExclusive = 0;
