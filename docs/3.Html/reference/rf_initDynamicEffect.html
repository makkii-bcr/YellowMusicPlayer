命令リファレンス 

Ymplayer.initDynamicEffect

Midiの再生中に動的に変更できる各エフェクト項目をデフォルト値に戻します

Ymplayerクラスを new で生成したときに、このメソッドが自動的に呼び出されます
(コンストラクタで呼び出しているため)
※ このメソッドの後に、updatePlayInfo()を呼び出すことで、MusicPlayerに反映されます

書式

initDynamicEffect()
下記のように変数がデフォルト値になります
  • midiVolume = 127;
  • midiTempo = 1;
  • midiPitch = 0;
