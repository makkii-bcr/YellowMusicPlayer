Ymplayer.initAllPlayingInfomation

Ymplayer.initAllPlayInfo

Midi, Ogg の再生オプション・エフェクト情報全てをデフォルト値に戻します

Ymplayerクラスを new で生成したときに、このメソッドが自動的に呼び出されます
(コンストラクタで呼び出しているため)
※ このメソッドの後に、updatePlayInfo()を呼び出すことで、MusicPlayerに反映されます

書式

initAllPlayingInfomation()

initAllPlayInfo()
このメソッドを呼び出すと、下記の関数をまとめて呼び出します
  • initDynamicEffect()
  • initStaticEffect()
  • initPlayingOption()
  • initDynamicEffectOgg()
  • initStaticEffectOgg()
