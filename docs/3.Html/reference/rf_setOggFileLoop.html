Ymplayer.setOggFileLoop

OggファイルのOggループを使用するかON/OFFを設定します。
OggファイルがOggループ(RPGツクール用ループ)に対応している場合は、そのループを使います。

※ このメソッドの後に、updatePlayInfo()を呼び出すことで、MusicPlayerに反映されます

書式

setOggFileLoop(fileLoop)
fileLoop
OggファイルがOggループ(RPGツクール用ループ)に対応していて、そのループを使うなら1、使わないなら0
※ 対応していなかったら通常のループになるが、fileLoop が 1 なら setOggloopStartTime(), setOggloopEndTime() の設定は無視される
