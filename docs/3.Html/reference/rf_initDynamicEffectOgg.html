Ymplayer.initDynamicEffectOgg

Oggの再生中に動的に変更できる各エフェクト項目をデフォルト値に戻します

Ymplayerクラスを new で生成したときに、このメソッドが自動的に呼び出されます
(コンストラクタで呼び出しているため)
※ このメソッドの後に、updatePlayInfo()を呼び出すことで、MusicPlayerに反映されます

書式

initDynamicEffectOgg()
下記のように変数がデフォルト値になります
  • oggVolume = 127;
  • oggTempo = 1;
  • oggPan = 0;
