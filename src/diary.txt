
// 開発日記 //
2012/ 5/ X : YellowMidiPlayerを作り始める
2012/ 6/ X : 開発一時中断
2012/ 7/2X : 開発再開
2012/ 9/ 1 : Midiの実装が完成
2012/ 9/ 2 : Oggの実装開始
2012/ 9/ 9 : Oggの再生・テンポ・パンなど実装完了
2012/ 9/10 : 一時停止・再開を繰り返すとOgg音楽がリバウンドする（一部分を繰り返してしまう）バグを修正中…
2012/ 9/11 : バグ修正完了。再生時間が正確になった
2012/ 9/12 : MidiOggPlayerに対応
2012/ 9/13 : TonyuでOgg Vorbisを鳴らすサンプルに対応
2012/ 9/16 : Tonyu強制終了時に自動終了できない場合があったので修正
2012/ 9/17 : MidiとOggを同時に鳴らせるように仕様変更
2012/ 9/19 : Oggの時間取得機能の再生時間が、たまに一瞬8秒遅れた時間になるバグを修正
2012/ 9/22 : MusicPlayerの状態取得機能の設定値が1000の時、Oggが再生・停止・一時停止・再開をした場合に無反応だったのを修正
2012/ 9/24 : DirectMusicのON/OFFを起動中に切り替えることができるようにした
2012/ 9/26 : OggでRPGツクール用のループに対応させた
2012/ 9/28 : OggでRPGツクール用のループに対応させた関係で、再生時間の計算を組み直した
2012/ 9/29 : ループ始端と終端の間を小さくすると、再生時間が上手く計算されないので直した
           : MidiOggPlayerモードでMidiの挙動がおかしかったので修正
2012/ 9/30 : Oggの初期の音量,テンポが0だったので、eff_midi.txtファイルが無いとOggが聞こえないバグを修正
           : eff_midi.txtが無い時、Midiを再生しない限りeff_midi.txtの有無を確かめないことで不具合があったので修正
2013/ 1/ 4 : Windows98でも正常に動くようにした
           : dlsファイルの登録解除バグを修正
           : Tonyuプロジェクトを移動したとき前のプロジェクトのOggが鳴りっぱなしにならないようにした（ただし、現プロジェクトでMidiを再生した時に限る）
2013/ 5/26 : oggを使わない場合でも、起動時にogg_sig.datファイルが生成されて邪魔くさいので、ogg_sig.datファイルがないときは生成しないようにした
2013/ 6/11 : ver2.10に変更
2013/ 6/15 : Tonyu 1.23以降とTonyu 1.22以前用にプロジェクトを分けた
           : ogg_sig.datファイルをTonyu終了時に削除するようにした
           : Tonyuのファイル名が「Player.exe」「Tonyu.exe」以外でYellowMusicPlayerの設定もしてない時でも、プロセス残りしなくした
2013/ 6/22 : Midiの名前保存数を255から65535に変更
           : Midiキャッシュ機能を廃止
           : ディスプレイの解像度を取得できるようにした
           : 再生中のOggのファイル名を取得できるようにした
2013/ 6/24 : Midiの演奏状態が演奏開始時に一瞬0(停止状態)になる仕様を0にならないように変更
2013/ 6/26 : 起動時にeff_midi.txtファイルが無い場合、Midiの番号がずれるバグを修正
2013/ 9/ 4 : Joysticの接続状態機能を追加
2013/12/14 : ループ処理をマルチメディアタイマーから、通常のスレッドに変更
           : Tonyuの複数起動に対応（終了処理のみ対応）
2013/12/15 : Oggでテンポが速いと、音飛びする可能性のあるバグを修正
           : Oggのサウンドバッファ書き込みが80秒分遅れると、それ以上は再生時間に間に合わせることをあきらめる仕様に変更
2013/12/23 : プロセスの終了検知処理を変更
           : ループ処理の変更によって、Windows98で出てきたバグを修正
2013/12/26 : Tonyu 1.22以前用のプロジェクトを統合（#defineでTonyu 1.23と1.22用のどちらのプログラムを作るか選択できる）
           : DirectMusicをON/OFFを切り替えると、Midiの音量とテンポが初期値に戻されるバグを修正
2013/12/27 : 今まで、Sleep(1);はtimeBeginPeriod(1);を設定すれば1msだとを信用しきっていたが、
             Windows2000でやるとどうやらSleep(1);でも200ms以上になることが判明したので、
             Sleep(1);は1msだと信用しないプログラムに変更
2013/12/28 : Tonyu 1.22以前の開発環境版でも、YellowMusicPlayerを使うことができるようにした（ただし、プロジェクトを変更する場合はTonyuを再起動する必要がある）
2013/12/31 : Windows98でOggを再生した後にMidiを再生するとOggが停止してしまうバグを修正（98ではファイルパスが全て大文字になり、プログラム上で大文字小文字を区別してしまっていたために、この現象が起きていた）
2014/01/14 : Unicode対応のプログラムが作れるように少し改良（中途半端でできていない）
2014/01/20 : Windows 8.1でプロセス残りが起きるらしいので、プロセスの終了条件を修正
           : デフォルトでは、NT系OSでDirectMusicがONになるのを、Vista以降でONになるように仕様変更（XP以前でDirectMusicをONにすると音量が小さく聞こえるため）
           : DirectMusicがONの状態でMidiデバイスが開かない場合、OFFにして再試行する仕様に変更
           : Midiデバイスが開けなかった場合は終了していたのを、終了しないように変更
