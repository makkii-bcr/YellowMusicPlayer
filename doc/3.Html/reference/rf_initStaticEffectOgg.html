Ymplayer.initStaticEffectOgg

Oggの再生前にあらかじめ設定する各エフェクト項目をデフォルト値に戻します

Ymplayerクラスを new で生成したときに、このメソッドが自動的に呼び出されます
(コンストラクタで呼び出しているため)
※ このメソッドの後に、updatePlayInfo()を呼び出すことで、MusicPlayerに反映されます

書式

initStaticEffectOgg()
下記のように変数がデフォルト値になります
  • oggFadeInTime = 0;
  • oggFadeOutTime = 0;
  • oggStartTime = 0;
  • oggLoopStartTime = 0;
  • oggLoopEndTime = 0;
  • oggFileLoop = 0;
