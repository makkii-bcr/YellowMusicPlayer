Ymplayer.setOggStartTime

Oggの再生開始位置を変更します。
Oggを最初からでなく、特定の時間から再生したいときに使います。

※ このメソッドの後に、updatePlayInfo()を呼び出すことで、MusicPlayerに反映されます

書式

setOggStartTime(startTimestartTime)
startTime
再生開始する時間をミリ秒単位で指定(1～(再生するoggの総再生時間-1))、指定しないなら0
※ setOggLoopEndTime()で0以外を設定した時は、値を指定できる範囲は(1～loopEndTime)になります
