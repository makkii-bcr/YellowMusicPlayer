使ってみる「MusicPlayerの状態を取得する」

MusicPlayerから、「Midi,Oggの状態」「再生中のMidi,Oggの情報」「コンピュータ内の日付と時刻」
「ディスプレイの解像度」「OSのバージョン」「ジョイスティックの接続状態」が取得できます。
※ 白丸が実際に取得できる項目です。

  • Midiの状態
    • 現在Midiは、再生中か停止中か一時停止中か
    • 演奏中のMidi番号（MusicPlayerで割り振られた番号）
      • ↑プロジェクトでMidiを登録した順番に、0から番号が割り振られます。
        256個(255番)まで正常に割り振られますが、257個(256番)以降は全て255番になります。
        65536個(65535番)まで正常に割り振られますが、65537個(65536番)以降は全て65535番になります。(ver2.20から変更)
    • 演奏中のMidiの、現在のデータ内のテンポ
    • 演奏中のMidiの、現在のティック
    • 現在のMidiデータ内での演奏時間(ミリ秒単位)
    • Midiを再生してからのループ回数
  • 演奏中のMidiの情報
    • ベースカウント
    • 総ティック
    • 総演奏時間（ミリ秒単位）
    • 最初の発音ティック
    • 最初の発音の時間（ミリ秒単位）
    • 最後の発音ティック
    • 最後の発音の時間（ミリ秒単位）
    • CC#111の位置（ティック単位）
    • CC#111の位置（ミリ秒単位）
  • Oggの状態
    • 現在Oggは、再生中か停止中か一時停止中か
    • 再生中のOggのファイル名（拡張子(.ogg)も含む）
    • 現在のOggデータ内での演奏時間(ミリ秒単位)
    • Oggを再生してからのループ回数
  • 再生中のOggの情報
    • チャンネル数
    • サンプリングレート
    • 総再生時間（ミリ秒単位）
    • 総サンプリング数
    • Oggループの開始位置（ミリ秒単位）（Oggループに対応している場合）
    • Oggループの終了位置（ミリ秒単位）（Oggループに対応している場合）
  • コンピュータ内の日付と時刻
    • 現在の時刻「時」
    • 現在の時刻「分」
    • 現在の時刻「秒」
    • 現在の時刻「ミリ秒」
    • 現在の日付「年」
    • 現在の日付「月」
    • 現在の日付「日」
    • 現在の日付「曜日」(0:日,1:月,2:火,…,6:土)
  • ディスプレイの解像度
    • ディスプレイの解像度「幅」
    • ディスプレイの解像度「高さ」
  • OS(Windows)のバージョン
    • OSの種類（NT系か9x系か）
    • OSの「メジャーバージョン」
    • OSの「マイナーバージョン」
  • ジョイスティックの接続状態
    • IDが1または2のジョイスティックがPCに接続されているか
MusicPlayerの状態を取得する手順：
  1. setGetMode()で、MusicPlayerの状態取得機能をONにします。
  2. updatePlayInfo()で、状態取得機能をONにしたことをMusicPlayerに反映させます。
  3. getPlayerStatus()でMusicPlayerから情報をTonyu側で受け取ります。
    ※ MusicPlayerの状態取得機能をONにすると、MusicPlayerがファイルの書き込みをします。
  4. getPlayerStatus()のページにある、get～()を使うと実際に情報が取得できます。

使い方は、以下のページを参照してください。
  • setGetMode()
  • getPlayerStatus()
