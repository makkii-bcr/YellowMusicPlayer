使ってみる「一時停止」 (Midi)

ここからは、YellowMusicPlayerの機能になります。
一時停止と一時停止からの再開をします。

pause()で一時停止、restart()で一時停止からの再開ができます。

[Main.tonyu]
extends SpriteChar;

$mplayer = new Ymplayer(); // $mplayerをYmplayerで上書き
while (1) {
  if (getkey(90) == 1) $mplayer.play($se_test ,1); // Z:再生
  if (getkey(88) == 1) $mplayer.stop();            // X:停止
  if (getkey(67) == 1) $mplayer.pause();           // C:一時停止
  if (getkey(86) == 1) $mplayer.restart();         // V:一時停止からの再開
  
  update();
}

pause(), restart()と、メソッドを呼び出すだけで使えます。

ちなみに、pause(),restart() は MidiとOggの両方に対して、命令が実行されます。
pause(1),restart(1) にすると、Midiに対してのみ命令が実行されます。
